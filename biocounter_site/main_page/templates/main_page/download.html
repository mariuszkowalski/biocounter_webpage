Instllation guides

Windows Python version

This installation guide is for Windows machines only. Due to differences in module names and implementation in mouse support BioCounter in current version will not work properly in the other operating systems.

  1. Download and install Python in the version at least 3.4.4 from official page.
  2. Run windows console aplication (cmd). Look for it in the start menu or mouse an keyborad shortcut, holding "windows" key press "r" button, in pop up windows type:
    cmd
    Next press "Enter" key or press "ok" button.
  3. In the console window type:
    python -m pip install --upgrade pip
    This is going to assure You will have the latest version of the Python package manager installed.
  4. Next in the console window type:
    pip install pillow
    This command is going to install the library required for Bio Counter to work.
  5. If You have not done it yet, download and unzip Python version of the Bio Counter.
  6. If everything went without any errors Bio Counter should lounch after double clicking of the "run.py" file in the main directory.
If You have installation problems please refer to detailed installation guides in the Support section of the page.
