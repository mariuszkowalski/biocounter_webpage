Features

The list of main features of the Bio Counter and biref instruction how to use it.

  • Load any photo or image into the program in the most common formats.
  • Count any objects by placeing an markers in the image.
  • Look for the counts in the statistics tab.
  • Export the image containing all placed markers.

Image load

Load and save image files containing any elements to count.

  • Supported file formats: JPEG, PNG and TIFF.
  • Choose the quality of exported images.

Markers managemet

Place up to 4 different markers, containing up to 8 group.

  • Change description and color of any marker group.
  • Change description of any marker.
  • Increase or decrease marker size to assure best visibility.

Statistics view

Look through statistics tab to quickly check counts and percentages.

  • Export all analytics data as CSV file, to be easily open in any spreadsheet software for further analysis.

Other features

Biocounter is small but contains other features to increase work flow.

  • Image zoom capability.
  • Shortcuts to make the work much faster.
  • Ability to set window in "always on top" mode.
