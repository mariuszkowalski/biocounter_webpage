Manual

Short instruction how to use the software.
The main program window have four main parts:

  • Menu bar at the top
  • Image area at the center
  • Status bar at the bottom
  • Tab window at the right side

Menu bar

Menu bar has been divided into three parts:

  • File - allows to oppen and close files.
    Keyboard shortcuts:
    • Ctrl + O - to open a file
    • Ctrl + S - to save processed file
  • Window - contains options to change window size and to enable "Always on top" mode.
    • Keyboard shortcut:
    • Ctrl + T - to togle "Always on top" mode
  • About - allows to show pop up window containing information regarding the software.

Image area

Image area takes the most space of the program window. The loaded images are placed in this area and this is only part of the window where placeing of a marker is allowed. This area also allows for zooming in and out of an image. If the image size is bigger than designated space aproperiate scrollbars will appear to at the bottom or the right side of the Image area.

  • Lleft mouse button - to place active marker in the image.
  • Right mouse button - near placed marker to remove it from the image.
  • Mouse wheel - to scroll the image vartically
  • Shift + mouse wheel - to scroll the image horizontally.
  • Ctrl + mouse wheel - to use the zoom function.

Status bar

Status bar shows few information regarding working of the aplication. The option has been inplemented but has not been used extensivly.
For example when marker size is changed using the keyboard shortcut, the current marker size is shown at the status bar.

Tab window

Tab window is part of the window containing all necessary tools and settings for those tools. It has been divided into three tabs:

  • Markers tab.

    Markers tab consits of 8 exact parts. Each part contains: sample name entry box, four switches for marker activation and indicator of the current marker color. Marker activation switches corresponds to qualifiers and given qualifier is common for all different samples. Description of the qualifier is adjustable. If the description length of qualifier exeds 3 letters, the shortened description will be presented in the button, but export of the statistics will have full description included.

    • To change sample description simply click left mouse button inside the field and type new name.
    • Pressing the qualifier button with left mouse button activates given marker. The button acts as switch and the marker stays activated until any other activation button is pressed. The marker will be still activated, even after going to another tab. After that it is ready to place in the canvas.
    • To change color of given marker press color label with left mouse button. Color dialog box will pop up. After picking new color every marker from this set will update its color and new color will be automaticly applied to new markers.
    • To clear all markers press "Clear all markers" button. Using this options clears also all the statistics. There is no confirmation window for that option and there is no undo option in the program.

  • Statistics tab.

    • Statistics tab is divided into 8 parts, each for the different marker. Those part has been divided into rows and columns, to create grid structure.

    • Header section of contains shortened qualifier descriptions separated into columns and description for the sum column.
    • Each section of the statistics tab corresponds to different sample. First row is labeled as "S + number" which indicates given sample. The numbers in the row corresponds to the numbers of placed markers for each qualifier, sum of the all qualifiers is shown as the last number. Second row is labeled "%" which indicates percentage of the corresponding qualifier quantity, shown directly above.
    • To export all acquired quantities and statistics press "Export as *.csv" button. An export dialog box will pop up. Choose folder and file name for exported file. The CSV file can be open with any office spreadsheet software and go under further analysis.

  • Options tab.

    • Options tab contains additional options for the qualifiers and markers.

    • Qualifiers section contains four different entry fields, each for every qualifier. The description of any qualifier may be changed here.
    • Dummy buttons has been placed to show a description preview. The button label is limited to three characters and if longer qualifier name is provided, the description will be shortened. Period at the end of the button label is an indication that description had been shortened.
    • In the markes size section there are buttons to decrease and increase size of all markers. The default value is set to 10. To decrease or increase the size press corresponding button using left mouse button. The minimal size of marker is 4 and maximum is 20.
      Keyboard shortcuts:
      • Ctrl + down arrow - to decrease size of marker
      • Ctrl + up arrow - to increase size of marker


Detailed installation guides

Windows Python version

This detailed installation guide contains step-by-step procedure for very beginner users, which would like to use Python version of the Bio Counter.

  1. Download and install Python in the version at least 3.4.4 from official page.
    From the "Downloads" tab choose "Windows". Choose latest release or at least Python 3.4.4 and download "Windows x86 executable installer".
  2. Run windows console aplication (cmd). Look for it in the start menu or mouse an keyborad shortcut, holding "windows" key press "r" button, in pop up windows type:
    cmd
    Next press "Enter" key or press "ok" button.
    Console window should appear.
  3. In the console window type:
    python -m pip install --upgrade pip
    Press "Enter".
    This is going to assure You will have the latest version of the Python package manager installed.
  4. Next in the console window type:
    pip install pillow
    Press "Enter".
    This command is going to install the library required to Bio Counter work.
  5. If You have not done it yet, download and unzip Python version of the Bio Counter with Your favorite archive manager.

  6. If everything went without any errors Bio Counter should launch after double clicking of the "run.pyw" file in the main directory.
